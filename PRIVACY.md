# Privacy Policy

## Dual Monitor Tab Move

This extension does not collect, store, transmit or share any personal or browsing data.

All functionality is performed locally within the user’s browser. The extension does not use analytics, tracking, external servers or remote code execution.

No user data is sold, transferred or used for purposes unrelated to the extension’s single function.

If you have questions about this policy, please contact the developer via the Chrome Web Store listing.
